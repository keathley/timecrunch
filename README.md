# Timecrunch

A simple timer app.

## Installation

Add this line to your application's Gemfile:

    gem 'timecrunch'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install timecrunch

## Usage

If you just want to start a timer you can run:

    $ tc start MINUTES
